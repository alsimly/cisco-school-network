📖 Overview

Cisco School Network is a network simulation project developed using Cisco Packet Tracer. The project demonstrates the implementation of a school-wide network infrastructure consisting of multiple departments, automatic IP allocation using DHCP, wireless connectivity through Access Points, and successful end-to-end communication testing.

✨ Features
✅ School-wide network topology
✅ Multiple subnet implementation
✅ DHCP Server Configuration
✅ Wireless Access Point Configuration
✅ Automatic IP Address Assignment
✅ End-to-End Connectivity Testing
